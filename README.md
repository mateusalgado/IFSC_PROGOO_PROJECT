# Calculadora Científica iPhone

## Simulação da calculadora do iPhone em Qt6/C++ com modos Normal e Científico

> Repositório destinado para o desenvolvimento do projeto final da disciplina PRG22107 - Programação Orientada a Objeto.
>
> Aluno: Mateus Salgado Barboza Costa
> Professor: Hugo Marcondes

---

### Descrição

Este projeto implementa uma calculadora inspirada na calculadora nativa do iPhone, com suporte a **modo normal** e **modo científico**, desenvolvida em **C++** com interface gráfica em **Qt6 Widgets**. O foco está na aplicação de conceitos de **Programação Orientada a Objetos**: herança, polimorfismo, encapsulamento, sobrecarga de operadores e uso da STL.

**Tecnologias utilizadas:**
- Linguagem: C++17
- Framework de interface: Qt 6.4
- Sistema de build: CMake 3.28
- Diagramas UML: PlantUML
